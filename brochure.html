About Xinchen Technology

公司简介 · Company Profile

让中国工厂也用得起的 AI 工厂大脑

鑫晨科技有限公司 (Xinchen Technology Co., Ltd.) 创立于 2026 年， 是一家专注于"AI 视觉质检 + 跨境出海智能"的科技型创业公司，总部位于浙江·杭州。

我们由资深 AI 视觉算法工程师与资深外贸运营人共同创办， 致力于将业界溢出的 AI 能力翻译成外贸老板用得起、看得懂、立刻能用的产品。 目前已落地 XinchenAOI 通用视觉质检平台、XinchenCopilot 出海智能 Copilot 与 XinchenCloud 数据中台三条产品线。

我们相信，中国制造业最大的杠杆不是更深的算法、更大的模型， 而是把"用得起"这件事做到位 —— 让每一家中小工厂都拥有自己的 AI 同事。

合作模式 · How We Work

每个项目都不一样 · 按需评估报价

我们不卖标准 SaaS 订阅。每个项目根据业务复杂度、数据量、行业要求联合评估，签 SOW 后启动。下面是常见的 4 种合作规模与典型周期。

CASE A

POC 评估

1–2周

探索可行性 / 业务方案验证

  • 需求调研工作坊
  • 技术可行性论证
  • 1 个 Demo / POC
  • 项目路线图建议
最常见 CASE B

MVP 共建

4–8周

单产线 / 单流程 MVP 落地

  • 定制开发 + 联调
  • 1 条产线上线
  • 3 个月免费迭代
  • 团队培训交接
CASE C

全量交付

3–6月

跨产线 / 多客户 / 持续运营

  • 多模块全栈交付
  • 对接 ERP / WMS / MES
  • SLA + 专属客户成功
  • 1 年迭代支持
CASE D

集团定制

6月+

集团 / 政企 / 长期战略合作

  • 多工厂 / 多区域部署
  • 本地化 / 私有化
  • 定制大模型 + 数据中台
  • 培训陪跑 + 长期驻场
定价说明： 每个项目签订前提供详细 SOW，按场景报价。无标准 SaaS 月费 — 我们不卖席位，卖结果。

Customer Outcomes & Solutions

客户场景 · Use Cases

三个核心场景 · 全链路覆盖

CASE 01

出货前质检

客户：中国外贸工厂

痛点：海外品牌方收货端检出 NG 后扣款 / 退货，损失大

解决方案

  • 桌面或流水线 AOI 工位部署在出货前
  • 不合格自动拦截 + 多语种报告同步给品牌方
  • 批次良率上传 XinchenCloud 数据中台
CASE 02

海外仓退货分拣

客户：海外仓 / 海外品牌方

痛点：FBA 退货堆积，人工分拣良品/瑕疵品成本高

解决方案

  • 退货分拣台：拍照工位 + 视觉模型自动判级
  • OK 品自动入库 / NG 品自动开退货报告
  • 对接客户 WMS，闭环处理
CASE 03

跨境出海运营

客户：外贸公司 / 跨境品牌

痛点：询盘量大、合规复杂、详情页质量差、买家良莠不齐

解决方案

  • XinchenCopilot 接管询盘、HS Code、合规扫描
  • 详情页 / SEO 关键词多语种自动生成
  • 买家背调 + 跟进序列，提升成单率
