# 鑫晨科技 · 产品矩阵 (Product Matrix)

> 为外贸工厂与海外品牌方提供 AI 视觉质检 + 出海智能 Copilot 的垂直方案商

## 产品线 1: XinchenAOI 通用视觉质检平台

| 形态 | 说明 | 价格 |
|------|------|------|
| **AOI Core SaaS** | 浏览器版，订阅制，按工位/月收费 | 499 元/月起 |
| **AOI Edge Engine** | 工厂私有部署，离线版，含 ONNX Runtime | 12,800 元/年起 |
| **AOI Pro On-Prem** | 大型企业版，含定制训练、多产线管理 | 项目制 |

## 产品线 2: XinchenCopilot 出海 AI Copilot

### 核心模块
| 模块 | 解决什么 | 输入/输出 |
|------|---------|---------|
| **询盘速回 (Inquiry Reply)** | 海外询盘 24h 内自动起草专业回复 | 询盘邮件 → 多语种报价回复 |
| **HS Code 助手** | 报关编码自动匹配，多国税则 | 产品中文/英文 → 6/8/10 位 HS code + 适用国家 |
| **合规雷达 (Compliance Radar)** | 标签/认证合规自动核查 | 产品图片 + 目标市场 → CE/FCC/RoHS/CCC 风险报告 |
| **详情页生成器** | 平台/语种适配的详情页 | 产品资料 → Amazon/Shopify/eBay 详情页 (EN/ES/PT/DE) |
| **买家背调 (Buyer Profile)** | 询盘买家可信度评估 | 公司名/邮箱 → 公司画像 + 风险标签 |
| **私域 RAG 知识库** | 公司产品/工艺/SOP 内化为可查询 AI | 上传文档 → 多语种问答 + 引用 |

### 价格
| 套餐 | 月费 | 适合 |
|------|------|------|
| **Solo** | ¥299/月 | 个人 SOHO 外贸 |
| **Team** (5 席) | ¥1,599/月 | 小型外贸公司 |
| **Factory** (15 席 + RAG 100GB) | ¥4,999/月 | 工厂出海部 |
| **Enterprise** | 项目制 | 集团客户 |

## 产品线 3: XinchenCloud 数据中台

把 AOI + Copilot 的数据汇聚到统一中台，输出 BI 看板和工单流。

| 能力 | 说明 |
|------|------|
| **质检数据看板** | 良率趋势、缺陷分布、产线对比、客户对比 |
| **客户级 SLA 报告** | 给海外品牌方按周/月推送质检报告（多语种） |
| **缺陷工单流** | NG 自动开工单 → 班长 → 品控 → 闭环 |
| **API/Webhook** | 与客户 ERP / WMS / MES 打通 |

定价：¥1,200–8,000 元/月 (按工位 / 数据量)

## 不做什么

- 不做医疗影像 (牌照地狱，让梅清做)
- 不做工业机器人本体 (让宇树/节卡做)
- 不做通用大模型 (用 Claude/Qwen API)
- 不做海外仓本身 (做服务商)
- 不做面向 C 端 (无客户基础)
