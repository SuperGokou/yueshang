## еҝ«йҖҹејҖе§Ӣ

### е®үиЈ…
```bash
cd trade-copilot
pip install -r requirements.txt
cp .env.example .env       # еЎ«е…Ҙ ANTHROPIC_API_KEY жҲ– DASHSCOPE_API_KEY
```

### дёҖй”®зҙўеј•з§Қеӯҗж–ҮжЎЈ
```bash
python -m yueshang_copilot.rag.ingest data/seed/  --collection demo
```

### еҗҜеҠЁ API жңҚеҠЎ
```bash
uvicorn yueshang_copilot.api.main:app --reload --port 8000
# Swagger: http://localhost:8000/docs
```

### иҜ•дёҖдёӘ Agent
```bash
# HS Code Agent
curl -X POST http://localhost:8000/agents/hscode \
     -H "Content-Type: application/json" \
     -d '{"product_name_zh":"304 дёҚй”Ҳй’ўдҝқжё©жқҜ","product_name_en":"304 stainless steel insulated bottle","target_country":"US"}'

# иҜўзӣҳйҖҹеӣһ Agent
curl -X POST http://localhost:8000/agents/inquiry-reply \
     -H "Content-Type: application/json" \
     -d '{"inquiry":"Hi, can you make 5000pcs custom-printed coffee mugs by Aug?", "lang":"en"}'
```
