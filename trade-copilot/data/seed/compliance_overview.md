# 跨境合规速查 (示例种子)

## 欧盟市场 (EU)

| 法规 | 适用范围 | 关键要求 |
|------|---------|---------|
| **CE 标志** | 几乎所有进入欧盟的工业品 | 自我申明 + DoC，部分需 NB 公告机构介入 |
| **RoHS 2011/65/EU** | 电气电子产品 | 限制 10 项有害物质 (Pb/Hg/Cd/Cr6+/PBB/PBDE/4 邻苯二甲酸酯) |
| **REACH** | 所有化学品和混合物 | SVHC 物质 0.1% 限值通报；年用量 1 吨以上需注册 |
| **WEEE** | 电气电子产品 | 注册 + 标志 + 回收义务 |
| **Battery Directive 2006/66/EC** | 含电池产品 | 标志 + 重金属限值 + 回收 |
| **EMC Directive** | 电气电子产品 | 电磁兼容测试 |
| **LVD** | 50–1000V AC 或 75–1500V DC 电气产品 | 安全测试 |
| **RED** | 无线产品 | 无线频谱+EMC+安全综合 |
| **GPSR (2026 起)** | 所有消费品 | 须有 EU 责任人 (代理人/制造商) |
| **CSRD** (2025+) | 大型品牌方 | 倒逼全链路供应商可追溯 |

## 美国市场 (US)

| 法规 | 适用范围 | 关键要求 |
|------|---------|---------|
| **FCC** | 含无线/电磁辐射设备 | Part 15 自我申明或认证 |
| **UL** (非强制但市场要求) | 电气类 | UL 标志 |
| **CPSC** | 消费品安全 | CPSIA 含铅+邻苯，儿童产品需 CPC |
| **FDA** | 食品接触/医疗器械/化妆品 | 注册 + 标签 |
| **California Prop 65** | 销往加州的产品 | 警告标签（含已知化学物质） |
| **CARB** | 销往加州 | 复合木材甲醛 / VOC |

## 中国出口 (CN→海外)

| 法规 | 适用 |
|------|------|
| **3C (CCC)** | 国内市场，出口非必需 |
| **生态环境部出口许可** | 化学品、危险品 |
| **商品检验** | 法检目录商品 |
| **海关编码** | 8 位 |

## 其他重要市场

| 国家 | 主要认证 |
|------|---------|
| 英国 | UKCA (取代 CE)，UKNI |
| 日本 | PSE (电气)、TELEC (无线) |
| 韩国 | KC (电气)、KCC (无线) |
| 印度 | BIS (电子电气强制) |
| 巴西 | INMETRO、ANATEL (无线) |
| 沙特 | SASO、SABER 平台 |
| 阿联酋 | ECAS |
| 澳大利亚 | RCM (电气) |

## 含电池产品额外注意

- **UN 38.3** 锂电运输测试（航空/海运/陆运）
- **MSDS** 化学品安全数据表
- 美国 **UL 1642 / UL 2054**, 欧盟 **EN IEC 62133**, 中国 **GB 31241**

## 含无线产品额外注意

- 频段许可：美国 (FCC ID)、欧盟 (CE-RED)、日本 (TELEC)、中国 (SRRC)
- 蓝牙：BQB 列名

## 标签合规通用要求

- 原产国 (Country of Origin)
- 制造商 / 进口商信息
- 安全警示
- 回收符号 (WEEE 叉桶等)
- 多语种说明书 (欧盟通常需当地语言)
